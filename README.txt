iStore Navid — Release v1.0

ساختار انتشار
- install.html: صفحه تیره نصب پروفایل؛ این لینک را برای مشتری ارسال کنید.
- index.html: صفحه روشن دسته‌بندی‌ها؛ آیکون نصب‌شده روی Home Screen این صفحه را باز می‌کند.
- category.html: فهرست وب‌اپ‌های هر دسته.
- iStoreNavid.mobileconfig: پروفایل نصب آیکون iStore Navid.

انتشار روی GitHub Pages
1) یک مخزن با نام iStoreNavid در حساب istorenavid بسازید.
2) تمام محتویات همین ZIP را در ریشه مخزن آپلود کنید، نه پوشه والد را.
3) GitHub Pages را از Branch اصلی و مسیر Root فعال کنید.
4) صفحه نصب مشتری:
   https://istorenavid.github.io/iStoreNavid/install.html
5) آدرس صفحه اصلی داخل پروفایل:
   https://istorenavid.github.io/iStoreNavid/

نکته مهم
اگر نام حساب GitHub یا نام مخزن را تغییر می‌دهید، قبل از انتشار مقدار URL داخل فایل iStoreNavid.mobileconfig را نیز متناسب با آدرس جدید تغییر دهید.

سیاست لینک‌ها
- اولویت با PWA و وب‌اپ رسمی است.
- در نبود PWA، فقط نسخه اینترنتی رسمی و مناسب موبایل استفاده شده است.
- هیچ لینک APK، IPA یا فروشگاه اپلیکیشن در فهرست قرار ندارد.
