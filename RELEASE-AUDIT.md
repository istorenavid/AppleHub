# iStore Navid Release v1.0 Audit

- صفحه نصب از صفحه دسته‌بندی جدا شد.
- لینک مستقیم وب‌اپ بانک ملت به `mobileweb.bankmellat.ir` اصلاح شد.
- مسیر پروفایل به `https://istorenavid.github.io/iStoreNavid/` تغییر کرد.
- شناسه‌های قدیمی پروژه از mobileconfig و فایل‌های متنی حذف شدند.
- PWA پایه برای صفحه اصلی افزوده شد: manifest و service worker.
- فهرست سفر با جاباما، اقامت۲۴، اتاقک، هومسا، فلای‌تودی، سفرمارکت و اسنپ‌تریپ تکمیل شد.
- فایل‌های JavaScript از نظر syntax بررسی شدند.
- فایل mobileconfig با plist parser بررسی شد.
- ارجاعات محلی HTML/CSS/JS بررسی شدند.
