# iStore Navid — Release checkpoint

## مسیر نهایی کاربر
1. کاربر `install.html` را باز می‌کند.
2. روی «نصب پروفایل» می‌زند.
3. پروفایل iStore Navid در تنظیمات iOS نصب می‌شود.
4. آیکون iStore Navid روی Home Screen ایجاد می‌شود.
5. آیکون، `index.html` را باز می‌کند و دسته‌بندی‌ها نمایش داده می‌شوند.

## الزامات نهایی
- برند یکپارچه: iStore Navid
- اولویت لینک‌ها: PWA، سپس نسخه وب رسمی
- بدون APK، IPA یا لینک مارکت
- صفحه نصب فاقد مرور دسته‌بندی‌ها است
- صفحه اصلی پس از نصب شامل پروفایل فروشگاه، تماس‌ها و دسته‌بندی‌ها است
